# Fanyatest

An online exam proctoring application that monitors students during tasks to ensure academic integrity.

## Features

### Student Authentication
- Secure student login system to verify identity before starting tasks

### AI-Powered Cheating Detection
- Uses AI to analyze student behavior in real time and detect signs of cheating
- Automatically captures a photo of the student when cheating is detected

### Anti-Cheating Measures
- **Paste Prevention** — Blocks clipboard paste actions to prevent students from copying answers from external sources
- **Tab-Switch Detection** — Detects when a student switches tabs or leaves the exam window and flags the activity

### Photo Capture on Violation
- Takes a picture of the student via their webcam when cheating behavior is detected
- Stores captured images as evidence linked to the student's session
